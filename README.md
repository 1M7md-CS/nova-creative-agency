# Nova Creative Agency

A modern, responsive website for Nova Creative Agency - crafting innovative web experiences and digital solutions.

## Live Demo

[https://1m7md-cs.github.io/nova-creative-agency/](https://1m7md-cs.github.io/nova-creative-agency/)

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across all devices
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions
- **Mobile-Friendly**: Optimized for mobile devices with hamburger navigation menu
- **Contact Form**: Functional contact form integrated with Formspree
- **Social Integration**: Linked social media profiles
- **SEO Optimized**: Meta tags and structured data for better search visibility

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with animations and transitions
- **Font Awesome**: Icon library for UI elements
- **Google Fonts**: Poppins font family for typography
- **Formspree**: Backend for contact form handling
